# 🚀 Projeto: Copiloto de Vendas com IA (Desafio DIO)

Este repositório documenta a criação de um **Copiloto de Vendas Inteligente** desenvolvido inteiramente através de **IA Generativa (Vibe Coding)**. O objetivo foi transformar a atividade proposta pela DIO em um MVP funcional utilizando prompts estruturados.

## 📸 Demonstração do Projeto
![Interface do Copiloto](./screenshots/dashboard.jpg)
*Interface moderna com foco em UX mobile-first e lógica comercial.*

## 🧠 Engenharia de Prompts (A Lógica)

O projeto foi construído utilizando os seguintes pilares de prompts:

### 1. Estruturação de Interface
> "Crie um aplicativo de 'Sales Copilot' para vendedores de loja. Estética Glassmorphism, Dark Mode, tons de azul e roxo. Inclua campos para transcrição e botões de ação rápida para Pitch, Objeção e Follow-up."

### 2. Configuração do Agente de Vendas
> "Atue como um Especialista em Vendas. Para o botão 'Próximo Passo', gere mensagens de WhatsApp usando o contexto do atendimento. Use gatilhos de escassez e o método de conexão humana."

## 🛠️ Ferramentas Utilizadas
- **Lovable.dev**: Para geração da interface e lógica React.
- **ChatGPT/Gemini**: Para refinamento da estratégia de vendas.
- **Prompt Engineering**: Técnica principal para construção do software.

## 📈 Resultados
- **Produtividade:** Criação de um app funcional em poucos minutos.
- **Assertividade:** Respostas padronizadas com técnicas reais de vendas (Feel-Felt-Found).
